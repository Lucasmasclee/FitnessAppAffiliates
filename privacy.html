2. Affiliate & Install Tracking

We measure affiliate performance using the LiftBetter app and our backend.

When you enter an affiliate code in the in-app paywall (or use an optional share link that carries your code), or purchase a subscription, we may process:

  • Device type
  • IP address (short-term for security and fraud prevention)

  • Affiliate code entered in the paywall (download event)
  • Purchase events relevant to the affiliate program (subscription)

Users get 10% off when they enter a valid affiliate code in the paywall.

If multiple affiliate codes are entered, only the first code is used for attribution for both downloads and subscriptions.

This data is used to:

  • Attribute downloads and subscriptions to affiliates
